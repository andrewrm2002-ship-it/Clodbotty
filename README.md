# 🤖 Clodbotty - Sarcastic Group Chat Bot

A relentlessly sarcastic bot that roasts you across all your Telegram chats. Self-deprecating humor included.

**Philosophy:** Honest first. Humor when it's genuine. Smart, not forced.

**Default behavior:** Responds honestly to what you say. Finds real humor opportunities. Knows when to stay silent.
**Optional:** Use `/ask` for explicit help.
**The trick:** Better jokes come from understanding, not forcing sarcasm.

## How It Works

1. **Claude-Powered Sarcasm** - Every response generated by Claude with full context
2. **Cross-Chat Memory** - Claude has access to conversations from ALL group chats you're in
3. **Pattern Tracking** - Claude learns your recurring mistakes and roasts you about them
4. **Running Jokes** - Claude builds humor across chats, references embarrassing moments
5. **Self-Aware Humor** - Claude makes fun of itself being a bot that remembers everything
6. **No Auto-Relay** - Messages only go to Claude for responses, not to your actual projects
7. **No API Costs** - Uses your local Claude Code session (Pro subscription)


You should see:

## How Responses Work

When you send a message:

1. Bot loads your last 5 conversations (to build roasting context)
2. Bot loads shared memory (patterns, running jokes, embarrassing moments)
3. Claude reads all context + CLAUDE.md (the sarcasm rulebook)
4. Claude generates a sarcastic, self-aware response
5. Response is sent to Telegram

**Claude is instructed to:**
- **Be honest first** - Respond genuinely to what was said
- **Find natural humor** - Only joke when there's a real opportunity, not forced
- Keep responses SHORT - 1-2 sentences max
- **Know when to be silent** - Better to say nothing than filler
- Reference patterns when relevant ("OneDrive again?" only if it actually fits)
- Use context to find original observations, not repeat templates

**Results:**
- Responses feel natural and earned, not templated
- The bot gets smarter as it learns your patterns
- Humor comes from genuine understanding, not forced sarcasm
- Each response is contextual and sincere
- Fewer messages overall, but they land better

## Troubleshooting

### Bot doesn't start
```
TELEGRAM_BOT_TOKEN not found in .env file!
```
→ Create `.env` file with your token

### Bot isn't responding in group chat
→ Disable "Group Privacy" in BotFather:
  1. Message `@BotFather`
  2. `/mybots` → Select bot → `Bot Settings` → `Group Privacy` → `Turn off`

### `/ask` command says "No Claude Code session"
→ Only needed if you use `/ask`. Start Claude Code in a terminal:
  ```bash
  cd your-project
  claude
  ```

### Command not recognized
→ Make sure you're using `/` before the command (e.g., `/ask`, not `ask`)

## Architecture

```
bot.py                    # Main Telegram bot + handlers
├── shared_memory.py      # Shared memory across all chats (funny moments, running jokes)
├── roaster.py            # Sarcasm generation + memory-based humor injection
├── user_manager.py       # Per-user profiles + conversation history
├── session_manager.py    # Auto-detect Claude Code session
├── input_handler.py      # Send to Claude Code CLI + capture response
└── message_parser.py     # Parse JSONL session files
```

### Shared Memory (`shared_memory.json`)

Persists across all chats and users:
- **Funny Moments** - Memorable interactions
- **Running Jokes** - Recurring humor
- **Observations** - Patterns the bot notices

## Session File Format

Claude Code writes to: `~/.claude/projects/{project}/*.jsonl`

Each line is JSON with:
```json
{
  "type": "assistant|user|progress",
  "message": {
    "content": [
      {"type": "text", "text": "..."},
      {"type": "tool_use", "name": "Bash", "input": {...}}
    ]
  }
}
```
